To modify game Initial Data, edit BubbleLevelConfigSO Scriptable Object.
To view grid cell position on the screen, open GridCell.prefab, then enable EditorMark.
To change bubble's flying speed, open Bubble.prefab, then modify FlyingSpeed field.
Enable GridSafeRectUI to view where the game would run inside, make sure this area to be 9:16 scale to prevent error.
